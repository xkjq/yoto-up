# Covers Tab Features

The new Covers tab in Yoto Up allows you to create and print covers for Yoto cards.

## Key Features

### 1. **Image Management**
- Add multiple images for covers (PNG, JPG, JPEG, BMP, GIF)
- View list of all added images
- Select and edit individual images
- Delete images from the list

### 2. **Image Editing Options**
For each image, you can configure:
- **Fit Mode**:
  - **Resize**: Resize to fit exactly (may distort aspect ratio)
  - **Crop**: Crop to fit (maintains aspect ratio)
  - **Scale**: Scale to fit (maintains aspect ratio, may have borders)
  
- **Crop Position** (when using Crop mode):
  - Center, Top, Bottom, Left, Right
  - Top-Left, Top-Right, Bottom-Left, Bottom-Right
  
- **Crop Offsets** (when using Crop mode):
  - Horizontal offset slider (-1.0 to 1.0)
  - Vertical offset slider (-1.0 to 1.0)
  - Allows fine-tuning of crop position

### 3. **Text Overlays**
Add text overlays to your cover images with:
- **Text content**: Enter any text
- **Font size**: 8-72 pixels (adjustable with slider)
- **Color**: Hex color with visual color picker
- **Position**: X and Y sliders (0.0 to 1.0, relative to card size)
- **Auto-update**: Changes to text, size, color, or position automatically update the preview
- Multiple text overlays per image

**Text Management**:
- List of all text overlays for selected image
- Click to select and edit
- Add, Update, or Delete text overlays

### 4. **Print Settings**
- **Paper Size**: A4, Letter, or Legal
- **Print Mode**:
  - **Overprint**: Print on top of existing card
  - **Underprint**: Print underneath (mirror image)
- **Margins**: Adjustable from 0-20mm
- **Cut Lines**: Toggle on/off for cutting guides

### 5. **Preview with Zoom**
- **Interactive Viewer**: Pan and zoom with mouse/trackpad
- **Zoom Controls**: 
  - Zoom In button
  - Zoom Out button
  - Reset Zoom button
  - Zoom level display (20%-500%)
- **Ctrl+Scroll Zoom**: Use Ctrl+scroll wheel for quick zooming
- **Smart Zoom**: Preview regenerates at appropriate resolution for zoom level

### 6. **Layout Optimization**
- Automatically calculates optimal layout to maximize covers per sheet
- Respects card dimensions (54mm Ã— 85.6mm)
- Adjusts for different paper sizes and margins

### 7. **Printing**
- Generate high-resolution (300 DPI) print layout
- Saves to temporary file
- Automatically opens in system default viewer for printing
- Shows location of saved file

## Card Specifications
- **Width**: 54mm
- **Height**: 85.6mm
- Cards are automatically arranged for optimal use of paper

## Workflow

1. **Add Images**: Click the "Add Images" button to select image files
2. **Edit Images**: Click on an image in the list to select it for editing
3. **Configure Fit**: Choose how each image should fit on the card
4. **Add Text** (optional): Add text overlays with custom positioning and styling
5. **Adjust Settings**: Configure paper size, print mode, margins, and cut lines
6. **Preview**: Use "Refresh" to update preview, zoom to inspect details
7. **Print**: Click "Print" to generate high-resolution output for printing

## Tips

- Use **Crop mode** with crop offsets for precise image positioning
- The color picker provides common colors for quick selection
- Text overlays auto-update as you adjust them - no need to click "Update" while editing
- Zoom in to verify text is readable and positioned correctly
- Use cut lines for easier cutting, or disable them for cleaner appearance
- Preview regenerates at higher quality when zoomed for better detail inspection
